## 剧情总结
在罗德岛重建的繁忙时期，可露希尔作为罗德岛的总工程师，承担着繁重的工作压力。故事开始于一场地块交接仪式，伦蒂尼姆的议长维娜将一座地块正式移交给罗德岛。在仪式上，可露希尔因走神而差点称呼错维娜的头衔，在博士的提醒下才得以纠正。这个小插曲显示出可露希尔已经工作到心神不宁的程度。

仪式结束后，阿米娅回忆起当初从雷姆必拓启程来维多利亚时，是可露希尔最先帮大家振作起来，用抢救出的巧克力和振奋人心的演说鼓舞了众人。可露希尔对此感到不好意思，用采购部涨价来威胁博士不要再提。博士表达了对可露希尔的关心，但可露希尔表示凯尔希已经把重担交给了她，她不会打退堂鼓。

工程部的Mechanist带来了为地块改造预拉的表格，厚厚一叠让可露希尔感叹，但她表示如果是自己来拉表格，会拉得更多。随后可露希尔开始处理接连不断的通讯。她先是接到关于零食网络渠道被芙蓉发现的消息，但让她意外的是芙蓉竟然放行了，这让她意识到不该用老眼光看待干员。

紧接着华法琳来访，告知医疗部的医疗用品储备和抗矿石病药物生产出现问题。可露希尔立即着手解决，查看各种原料的采购渠道。正在这时博士来电，已经帮可露希尔给纳斯尔纱发了信，解决了部分原料问题。可露希尔承诺会给博士送沙虫腿作为感谢。华法琳关心可露希尔的身体状况，询问她一天睡几个小时，可露希尔承认一天只睡四个小时还拆成四份，但表示这在卡兹戴尔时就是常态。

人事部的梓兰带来了几份简历，可露希尔快速处理了这些人事安排，将面试安排到一个半月之后。她决定要想办法解决睡眠问题，想出了一个利用支援机械帮助管理时间的方法。

可露希尔开始动员各个支援机械协助工作。Thermal-EX被设置了关键词捕捉单元，一检测到“华法琳”就会用爆炸提醒她。当Medic来汇报塔拉的原料批次规格不一致时，不慎触发了Thermal-EX的冲击功能。可露希尔解释了这个设置，随后就去医疗部处理原料问题。

在完成训练室相关工作后，可露希尔与Logos交谈。Logos表示并非催促她，而是想为她送上一句咒言帮助睡眠。可露希尔拒绝了，表示自己睡不着只是因为事情太多。话音刚落，连续四个通讯请求同时到来。Logos目睹可露希尔从身上接连掏出多个终端，像自己跟自己吵架一样应对着不同的问题。

可露希尔急中生智，让Castle-3启动自动回复模式，将七个终端全部接在Castle-3上，让它按轻重缓急排序通讯。她带着Castle-3匆忙去处理ZOOT系统报错的问题。

当可露希尔气喘吁吁地见到煌和澄闪时，澄闪敏锐地注意到可露希尔的黑眼圈。可露希尔用让澄闪帮她剪头发为借口搪塞过去，但煌指出了这一点。Lancet-2计算出可露希尔有机会进入睡眠的概率，而正义骑士号则让贾维沉迷游戏无法打扰可露希尔，提高了睡眠几率。

PhonoR-0告诉可露希尔，她每天不工作的时间平均只有五十六分钟。就在这时，所有支援机械陆续报告接下来的二十四小时内不会有新的通讯或需要处理的事务。虽然感觉奇怪，但疲惫至极的可露希尔顾不上多想，倒在行军床上就睡着了。

在梦中，可露希尔见到了特蕾西娅殿下。殿下赞许她的勤劳，告诉她传统的枷锁已经被突破。可露希尔恍惚中记起了过往，记得殿下第一次推开她的门，委托她重任的往事。殿下带她参观了已经完工的罗德岛新基地，包括精心设计的游戏室、堪比移动城市的动力层，以及医疗部。当看到医疗部还有问题时，可露希尔一度惊醒，但殿下用术法解决了所有问题，告诉她只需要点点头、签签字、按几个按钮就够了。

最后来到工程部，那里有殿下准备的礼物——罗德岛总工程师的宝座。殿下说从今以后可露希尔只需要坐在宝座上吃巧克力、薯片等零食。可露希尔虽然向往不工作，但总觉得这样不对。当她想要拒绝时，殿下说出了“以后再见了”。

可露希尔惊醒，发现自己睡了整整二十四小时。华法琳、煌、Medic、Logos、Mechanist等人都在场。原来在她睡着期间，博士、阿米娅和大家帮她分担了工作。博士重新规划了可露希尔的日程，让她一天能睡五六个小时。阿米娅表示会继续支持可露希尔。

可露希尔想再偷懒睡一会儿，却发现博士因为帮忙处理工作已经二十四小时没睡，直接倒在了地上。刚以为可以松一口气，Castle-3又带来了三个未接来电，分别来自工程部、医疗部和采购部。可露希尔发出绝望的叫声，新一轮的忙碌又要开始了。
