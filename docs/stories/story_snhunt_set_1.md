## 剧情总结
《射向群山的最后一箭》的故事发生在谢拉格，一场灾难性天灾发生一周后。整个活动围绕着主角茱安娜，一个虔诚的耶拉冈德信徒的女儿，在她父亲格里沙遇难后，试图理解父亲行为的真相，并在信仰与亲情之间寻求和解的内心历程展开。

故事伊始，谢拉格举行了一场庄重的遇难人员纪念碑揭碑仪式。在修士的祝祷声中，人们为逝者祈福，希望他们能在耶拉冈德的怀抱中与群山共眠。茱安娜也身处人群之中，她的思绪却回到了与父亲格里沙的过往。那些记忆片段勾勒出格里沙对女儿的深沉父爱，他教导茱安娜狩猎的技巧，引导她相信耶拉冈德的旨意，同时又对茱安娜的病痛感到担忧，甚至表示即使耶拉冈德未留下旨意，他也会替祂保证一家人同在，这为后文格里沙的“背弃”行为埋下了伏笔，暗示了他的行动可能源于对女儿的爱。

揭碑仪式后，茱安娜按照一封信的指引，来到一家礼品店寻找父亲留下的遗物。在店门口，她意外地发现了一支父亲的猎箭，这支箭并非寻常之物，上面绑着用于传递信息的箭筒。她从礼品店老板口中得知，格里沙在天灾前一周，突然决定不再供货，并匆匆将一些物品寄存在这里，其中包括一箱兽皮、一个包裹以及那把对格里沙来说极其重要的重弩——他从不让他人触碰的武器。这些反常的举动和留下的贵重物品，让茱安娜感到困惑：一个以打猎为生的人，为何会放弃自己的生计，甚至连最重要的武器都托付于人？

在地下室里，茱安娜再次与父亲的遗物相伴。她抚摸着承载着父女共同狩猎记忆的兽皮，感受着父亲生前的气息。然而，她的裂云兽却对其中一张兽皮表现出异常的反应，像是嗅到了“敌人的味道”。在裂云兽的指引下，茱安娜在最底层发现了一张绘有地图的光秃秃兽皮。这张地图的发现，立刻引起了礼品店老板的警觉。他透露，维多利亚军队进城前后，这类“违禁品”地图在城中多处被发现，持有者甚至被怀疑窝藏间谍。老板的言辞，以及他急于与这些“违禁品”撇清关系的举动，无疑是在暗示格里沙可能参与了与维多利亚人相关的行动，甚至可能是谢拉格的“背叛者”。

这一发现犹如晴天霹雳，彻底颠覆了茱安娜对父亲的认知。在她心中，父亲是一个对耶拉冈德信仰坚定不移的人，从未有过一丝怀疑，更不可能背叛谢拉格。然而，物证的存在和礼品店老板的含沙射影，让她感到无力反驳。她内心的声音，此时化作格里沙对耶拉冈德的质问：“可我的孩子，她的病情从未能好转，她的药量也不断增加，你为何将如此宿命降临在虔信之人的头上？”这段内心独白揭示了格里沙内心深处的挣扎，他为了女儿的病痛，甚至质疑耶拉冈德的公平性，并最终决定“倘若你的启示当真如此残酷，那么我......只好相信自己。”这句独白成为了理解格里沙行为的关键线索。

在沉重的打击下，茱安娜徘徊在纪念碑前，父亲的名字此刻显得异常刺眼。她感到愤怒，却也无法记恨父亲，甚至觉得自己没有资格去这么想，内心充满了自责和矛盾。她拿出父亲留下的箭，在纪念碑上父亲的名字处刻划，试图发泄内心的痛苦与不解。

就在她几乎崩溃之际，一群谢拉格孩子走上前来，打破了她的绝望。他们正在纪念碑上寻找一个叫“格里沙”的恩人。孩子们天真无邪的讲述，揭示了灾难发生时的真相：当车站即将坍塌，人们还在原地祷告时，是格里沙叔叔大喊着让他们不要待在原地，并冲上前将他们推出危险区域。他因此被倒塌的穹顶掩埋，消失在孩子们的视线中。大人们都说，格里沙先生回到了耶拉冈德的怀抱。

孩子们的证词，如同一道温暖的光芒，瞬间驱散了茱安娜内心的阴霾。她终于明白了父亲的“背弃”并非真正的背叛，而是在危急时刻，他做出了最符合耶拉冈德教诲的行动——挽救生命。他所谓的“与维多利亚人合作”，可能只是为了给病重的女儿筹集医疗费用，而他在灾难中的英勇举动，更是对生命的极致尊重和守护。他牺牲了自己的名誉，甚至可能被误解为背叛者，却拯救了无辜的生命。

在父亲的墓碑前，茱安娜听到了内心深处父亲的声音：“好孩子，爸爸有办法让你好起来的，你一定要相信我......我只能装作彻底抛弃耶拉冈德，才能摆脱与他们合作的悔恨，才能有钱为你治病。”这段“内心独白”彻底解释了格里沙的动机：他为了女儿的病，甘愿背负骂名，甚至“背弃”信仰的表象，但他的内心依然是虔诚的，愿意接受耶拉冈德的惩罚。他深知自己的选择可能带来误解和痛苦，但他为了女儿的生机，义无反顾。

茱安娜将父亲留下的箭扣在重弩上，向着群山的方向，射出了这支箭。这一箭，不再是猎物的宿命，而是她对父亲理解、接受与最终的宽恕。她不再为父亲的“背弃”向耶拉冈德致歉，因为她明白父亲践行了耶拉冈德的旨意——让生命得以延续。这支箭，象征着父女之间跨越生死与误解的沟通，也代表着茱安娜从迷茫走向坚韧，从痛苦走向和解的成长。她最终相信，即便耶拉冈德对她和父亲保持缄默，父亲的爱和他的英勇行为，本身就是耶拉冈德的旨意。她接受了父亲为爱所做的“挣扎”，并以自己的方式，完成了对父亲的“忏悔”的理解与回应。

这场活动剧情通过一个家庭的悲剧，探讨了信仰、道德、牺牲与爱的复杂关系。格里沙的故事，是一个普通人面对绝境时，为了挚爱而做出的艰难抉择，他的行为超越了世俗的评判，成为了一种更为宏大的牺牲。茱安娜的旅程，则是一个女儿如何从误解、痛苦中走出来，最终理解并继承父亲精神遗产的感人过程。整个故事虽然篇幅不长，却蕴含着深刻的情感和哲学思考，展现了明日方舟文案一贯的内敛而有力的叙事风格。
