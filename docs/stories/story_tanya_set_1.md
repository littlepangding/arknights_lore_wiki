## 剧情总结
在明日方舟活动“昨日已死”的幕间章节中，剧情聚焦于整合运动成员瑭雅在泽尔格勒疏散行动中的心理崩溃与自我救赎。故事以一场急促的演讲开场——瑭雅向居民揭露第一集团军的罪行，指出他们进行的实验导致怪病蔓延，并含泪宣告自己最小的妹妹妮娜因此丧生。她号召所有人立刻撤离这座即将被摧毁的城市。演讲起到了效果，居民们开始慌乱地寻求她的帮助，但瑭雅的精神状态早已摇摇欲坠。

起初，瑭雅展现出惊人的意志力：她挨家挨户敲门，帮助几十个家庭疏散，甚至背起迷路的孩子。然而，创伤后应激障碍迅速吞噬了她。当她试图帮助年老的路基娜女士时，后者因执意找寻先生送的假金耳环而拖延时间，最终在瑭雅去疏散其他居民后，路基娜从梯凳上摔落身亡。瑭雅撞开门发现尸体，陷入深深的自责。

随后，瑭雅的幻觉全面爆发。她先后见到已故的父母和妹妹的幻影，这些幻影最初以温暖的方式出现——父亲鼓励她，母亲安慰她，但瑭雅清醒地戳破了幻象：“爸爸妈妈，你们已经死了。”她回忆父亲曾是一名勇敢的消防员，后来却变成嗜酒如命的懦夫，最后醉死在火场；母亲因矿石病痛苦不堪，跳楼自杀前连一句“再见”都没留下。这些创伤让她产生了强烈的自我否定。

幻觉进一步恶化，化身成一个漆黑的幽灵，以死者的声音嘲讽她，声称她所有的努力不过是自我感动。幽灵甚至变成妮娜的样貌，指责她害死了妹妹。瑭雅在追捕幽灵的过程中闯入路基娜的家，发现一个趁火打劫的窃贼正掰开路基娜僵硬的尸体偷走那对假金耳环。理智彻底崩溃的瑭雅开始疯狂殴打窃贼，一边哭喊“为什么”，她无法理解为何有人能在如此灾难中仍行卑劣之事。

图林及时赶到，强行喝止了瑭雅。他看到鼻青脸肿的窃贼后，指出了真正的罪恶：“他做的事情是邪恶的，但真正的罪恶并不仅仅在于他。要怪那些把我们教导得如此愚蠢而自私的人。”瑭雅质问自己是否配得上“整合运动”之名，图林回答：“没有什么‘配不配得上’，一切都只取决于你自己，取决于你做了什么。”

在那一刻，瑭雅终于与心魔达成和解。她看见那个漆黑的幽灵在血液中消散，于是平静地为窃贼包扎伤口，说道：“我还是决定，要活下去。我还是要……救人。”她抱起窃贼如同抱起自己的弟弟妹妹，将其安置在伤员推车上，转身继续加入疏散工作。离开街道时，她的余光瞥见已故父母和长大后的妮娜的幻影站在空屋门前，向她投来关爱体谅的目光，但她忍住了没有回头。

整个剧情深刻展现了瑭雅如何面对自己的内疚、弱点和创伤。她曾试图用行动弥补过去，却几乎被幻觉吞噬；最终，她接受了“昨日已死”的事实，选择继续活在当下，用救赎性的行动而非自我毁灭来证明自己的价值。故事未提供完整的背景——瑭雅与整合运动的关系、第一集团军的具体实验、泽尔格勒的最终命运均未展开，但这段幕间成功刻画了一个在崩溃边缘重获力量的角色。
