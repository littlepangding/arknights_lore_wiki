## 剧情总结
在《明日方舟》活动“号外！号外！”的幕间章节中，故事以乌萨斯圣骏堡为舞台，围绕乌啾（一只活泼的鸟？实际上是一名罗德岛办事处的年轻成员）与好友烈夏、凛冬打赌经营报摊一周的历程展开。乌啾自信满满，认为自己作为前卖报童经验丰富，只要在一周内赚够十切尔文就能收支平衡，从而证明自己的能力，获得自由选择职业的权利（比如当罗德岛接待员）。然而，这场看似简单的经营挑战，却因一连串社会事件和人际纠葛变得异常沉重。

周一，晴。乌啾顺利卖出两份报纸：一位年轻议员因发现报纸上关于纺织工人抗议的新闻可以作为演讲素材而慷慨解囊；一位焦躁的市民因担心抗议影响自己公司的毛衣销售业绩而购买报纸。乌啾还遇到一位提着沉重包裹的仆役，她正寻找失散的“亲人”帕霍米，并许诺之后提供全名。乌啾心生同情，答应帮忙留意。

周三，阴。焦躁的市民再次光顾，抱怨晋升答辩中同事彼得与评委有不当交易，乌啾戏谑地建议用泻药报复，被拒绝。仆役冒险从男爵书房偷来帕霍米的全名（帕霍米·米哈伊洛维奇·戈里亚乔夫）写在劳动契约书上，并因口袋被缝无法藏东西，将契约书留给乌啾。乌啾注意到契约书有问题，但未说破。年轻议员鼻青脸肿地出现，得意地称被政敌报复是“斗士的勋章”，并解释工人抗议起因是机器停工导致强迫手工劳作却无加薪。乌啾好奇死伤者名单，议员离开后，乌啾从报纸上看到了名单。

周五，阴云密布。警察以“不实消息”为由销毁所有报纸，包括乌啾的存货。乌啾偷偷藏起一份报纸和契约书。焦躁的市民决定采纳泻药计划，但最终承认自己与工人处境相似：命运不掌握在自己手里。年轻议员宣布自己成功得罪了公爵（因嘲笑其脖子），并邀请其共进午餐，同时声称将推动立法禁止未成年人经营商铺，让孩子们读书。乌啾追问工人待遇问题，被警察打断。之后，一位平民女性替仆役转交报酬——几枚沾满泥、血和锈的硬币。乌啾追问仆役下落，得知她似乎“生病了”。乌啾检查契约书和报纸，发现所谓的“亲人”帕霍米实际上是用“卖身契”永久卖掉了仆役，且此人很可能已死于工人抗议事件。

周日，晴。表面恢复平静，但乌啾内心沉重。焦躁的市民升任小组长（彼得成了他的上司），却感叹自己与工人无区别。年轻议员失魂落魄地出现，公文包里被塞进血淋淋的鼷兽尸体和门把手，附警告字条“不许再嘲笑公爵大人的脖子了”。他意识到政敌在意的不是他的主张，而是他嘲笑了脖子，崩溃离开，连报纸钱都没付。乌啾算账发现正正好差一切尔文。这时，刻薄的贵族（男爵）来讹诈，声称仆役已死，按契约其财产包括那几枚硬币应归他所有。乌啾愤怒拒绝卖报纸给贵族，哪怕输掉赌局也要坚守底线。最终，乌啾遇到一位刚回圣骏堡的陌生人，将藏起的真相报纸以十切尔文高价卖出，并告诉他“在一座城市，有一群走投无腹的纺织工人……流了很多血”。陌生人严肃回应：“我现在知道了。”

剧情以乌啾的内心成长收尾：她明白了有些真相不能被掩埋，而一份报纸的意义不仅仅是新闻内容，更是连接人与人的命运。
## 剧情高光
1. **乌啾拒绝卖给贵族报纸**：“我不卖给你。我绝对，绝对，永远！不会卖给你！哪怕因此输掉赌局，哪怕我以后都不能再当接待员......我也不会卖哪怕一张废纸给你！”——这是全剧情绪最高点，展现了乌啾从天真卖报童到坚守正义的觉醒。

2. **年轻议员的崩溃时刻**：“脖子——脖子？我什么时候——不！我以为他不高兴是因为我的主张、我的提案！不！他在意的，只是我嘲笑了他的脖子？哈，哈！真好笑。”——理想主义者的幻灭，讽刺了乌萨斯政治的荒诞。

3. **乌啾发现契约书真相**：“那份‘契约书’完全是一张‘卖身契’。乌啾：帕霍米，她那么信任你，但你为什么——乌啾：我该怎么告诉她，是她一直寻找的那位失散的‘亲人’，亲手卖掉了她？而且期限还是，永久？”——残酷真相与仆役的信任形成尖锐对比。

4. **陌生人回应**：“我现在知道了。”——简短但分量极重，代表真相终会被传递，呼应主题“号外”。

## 角色剧情概括
-   **- 乌啾**：主角，为证明能力经营报摊。在接触各类人物和发现仆役被骗、工人抗议被镇压的真相后，从单纯想赢赌局转变为坚守正义，拒绝向压迫者妥协，最后高价卖出真相报纸。
-   **- 烈夏**：乌啾的同伴，提供了经营许可证和赌约条款，戏份较少，是引子角色。
-   **- 凛冬**：乌啾的同伴，提供了本钱和赌约，与烈夏共同设定了一周亏损则乌啾顺从的条件。
-   **- 年轻议员**：理想主义的改革者，试图通过演讲推动工人待遇改善，但政治对手只在意对他人的嘲笑而非主张。最终被恐吓手段击垮，体现了乌萨斯官僚体系的扭曲。
-   **- 焦躁的市民**：普通上班族，最初只关心个人晋升和业绩，后在乌啾点醒下意识到自己与工人处境相似，但依旧无力改变。
-   **- 仆役**：寻找失散“亲人”的可怜女性，实际被帕霍米以契约书永久贩卖。最终疑似因贵族苛责而死，成为乌啾觉醒的触发点。
-   **- 刻薄的贵族（男爵）**：压迫者的代表，纺织厂被砸后迁怒于人，逼死仆役后还来索要她的遗产，被乌啾拒绝卖报。
-   **- 陌生人**：结尾出现的神秘角色，刚从外地返回圣骏堡。从乌啾处高价购买真相报纸并倾听她的讲述，象征公众对真相的见证与接纳。
